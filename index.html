    Gerar/Gerenciar Etiquetas

    Nesta página você cria e mantém os layouts de etiquetas. É possível importar da biblioteca um layout existente para edição, digitar/colar o ZPL direto no editor, detectar os campos disponíveis do Teep para usar como placeholders (ex.: {OP}, {Quantidade}, {Maquina}), limpar o editor, visualizar o preview renderizado via Labelary e salvar o layout na Biblioteca. Em “Avançado” você ajusta o tamanho (polegadas) e a resolução do rótulo; essas definições são gravadas junto com o layout para futuros previews.
